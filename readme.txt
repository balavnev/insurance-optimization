в папках 3 набора: простой, средний, сложный

описание задачи
###############################################################
простой

constraint_low.txt - ограничение просто по штукам

sample_in_low.txt  -полное множество для оптимизации (формат пивотированный)
предлагаем 2 продукта осг и кск, в 2 канала смс и емэйл
сабжисн=ид клиента
формула ЕВ=премиум*проба*маржин -кост
надо макс ЕВ при заданных ограничениях

sample_out_low.csv - ответ нашего оптимизатора
subjisn=ид, 
п1п2п3п4 рассчитанный по формуле ЕВ для каждого предложения
оффер : 0 -не предлагаем ничего, 1/2/3/4 - конкретная выбранная опция

###############################################################
средний

constraint_med.csv - ограничения на задачу

sample_in_med.csv - полное множество для оптимизации (формат плоский)
сабжисн=ид клиента
предлагаем 3 продукта ифл, кск, кск_осг, в 3х каналах смс емэйл осрм, и много подсегментов
скор= он же премиум*проба*маржин
формула ЕВ=скор -кост
кост - смотри в ограничениях
надо макс ЕВ при заданных ограничениях

sample_out_med.csv - ответ нашего оптимизатора
subjisn=ид, 
yyyyyyyyy рассчитанный по формуле ЕВ для каждого предложения
x1...x15 - конкретная выбранная опция 1-берем ее, 0 - не берем

###############################################################
сложный

constraint_hard.csv - ограничения на задачу

sample_in_hard.csv - полное множество для оптимизации (формат плоский)
сколько-то продуктов, каналов, подсегментов
скор=он же проба
авг_чек= он же премиум
ормула ЕВ=скор*авг_чек*маржин -кост
кост, маржин - смотри в ограничениях
надо макс ЕВ при заданных ограничениях

sample_out_hard.csv - ответ нашего оптимизатора
расчитанные все показатели
опт =1 берем, 0 - не берем
